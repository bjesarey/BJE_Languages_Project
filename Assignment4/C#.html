Basic information

C# was created by Anders Hejlsberg, Scott Wiltamuth, and Peter Golde in 2000 for Microsoft's .NET framework. C# was designed to be simple, portable for C and C++ users, and supporting of software engineering principles. C# has had over 12 versions throughout its history, with the newest version upgrading syntax to be simpler and adding new parameters to help the code run faster. Today, C# is used for Windows applications and game development.
