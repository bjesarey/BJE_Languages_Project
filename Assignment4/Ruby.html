Basic information

Ruby was created by Yukihiro Matsumoto in 1995 because he wanted to create a language that was more powerful than Perl, and more object-oriented than Python. Ruby has had over three versions, with the newest version running three times faster than its predecessors with the help of just in time compilation. In Ruby, everything is object-oriented, meaning parts of the language can be changed to fit the need of the user. Ruby is used for web development, static site generation, automation, building web servers, and data processing. Thanks to Ruby on Rails, which is a Ruby written framework for building websites, multiple well known applications have been built, including but not limited to: GitHub, Hulu, and Airbnb.
